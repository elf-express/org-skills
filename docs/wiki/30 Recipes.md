# 食譜



> 針對常見技能分享場景的複製貼上解決方案。



每個配方都遵循相同的結構：**場景**（何時/為什麼）、**解決方案**（逐步）、**驗證**（如何確認）和**變化**（替代方案）。



## 可用食譜



|食譜|場景 |
| ---| ---|
| CI/CD 技能驗證 | GitHub Actions / GitLab CI 中的稽核 + 同步 |
|民營企業技能|使用令牌驗證安裝私人儲存庫 |
|專案模式工作流程|完整的專案範圍技能生命週期 |
|跨機器同步 |跨機器的推/拉技能 |
|團隊入職|新進員工技能環境設定 |
|集中技能儲存庫 |每個開發人員配置中的技能和目標的一個儲存庫 |
